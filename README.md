# 3D Portfolio - Mr-Saad

A modern, interactive 3D portfolio website showcasing projects, skills, and experience with cutting-edge web technologies.

## 🚀 Features

- **3D Interactive Hero Section** - Animated Three.js sphere with particle effects
- **Responsive Design** - Mobile-first approach with Tailwind CSS
- **Smooth Animations** - Framer Motion and GSAP for seamless transitions
- **Project Showcase** - Detailed project cards with technologies and highlights
- **Skills Section** - Organized by categories
- **Contact Integration** - Direct links to GitHub and social profiles
- **Dark Theme** - Modern slate color scheme with blue accents

## 🛠️ Tech Stack

- **Frontend Framework**: React 18 + TypeScript
- **3D Graphics**: Three.js + React Three Fiber
- **Styling**: Tailwind CSS + PostCSS
- **Animations**: Framer Motion
- **Build Tool**: Vite

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Mr-Saad1/3d-portfolio-mr-saad.git
cd 3d-portfolio-mr-saad

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
